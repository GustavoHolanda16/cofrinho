# cofrinho
Sobre o Projeto

Sistema desenvolvido em Java para a disciplina de Programação Orientada a Objetos (POO), implementando os conceitos fundamentais de herança e polimorfismo através de um cofrinho virtual que armazena diferentes tipos de moedas.

Objetivo da Atividade

Demonstrar a aplicação prática dos pilares da POO:

    Herança - Classes específicas herdam de uma classe abstrata

    Polimorfismo - Tratamento uniforme de diferentes tipos de moedas

    Encapsulamento - Proteção dos atributos das classes

    Abstração - Definição de contratos através de classes abstratas

Competências Desenvolvidas

    Modelagem de classes com herança

    Implementação de polimorfismo

    Manipulação de coleções em Java

    Desenvolvimento de interfaces de linha de comando

    Aplicação prática dos pilares da POO
